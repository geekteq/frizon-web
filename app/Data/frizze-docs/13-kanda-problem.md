# Kända Problem och Vanliga Fel

## Adria Twin 600 SPT - Vanliga Problem

### Läckage och Vattenintrång
- Takfogar kan ha defekter som leder till vattenintrång vid regn
- Kontrollera tätningar regelbundet, särskilt runt takluckor
- Kondens i skåpet ovanför förarhytten vid vinterkörning är vanligt - kan bildas istappar vid låga temperaturer

**Status 2026:** Främre takluckan har tätats. Inväntar kraftigt regn/skyfall för att verifiera att läckan är åtgärdad.

### Vikt och Lastkapacitet
- Maxvikt 3500 kg kan lätt nås med full utrustning
- Väg husbilen på våg innan längre resor
- SVtech kan uppgradera från 3300 kg till 3500 kg om behov finns

### Fjädring
- Hård fjädring är vanligt förekommande, särskilt vid tom bil
- Luftfjädring (Air Ride Suspension) kan installeras för bättre komfort (~5000-6000 kr)
- Knirkande ljud kan tyda på att fjädringen slår i botten även vid låg last

### Vattenpump
- Pumpen kan sluta fungera efter några sekunders drift
- Kontrollera säkringar - de kan vara svåra att lokalisera
- Elskåpet sitter vanligtvis under sängen eller i garderob

### Säng
- Gasfjädrar för uppfällning kan tappa kraft över tid
- Kontrollera att låsmekanismen fungerar korrekt

## Citroën Jumper 2.0 HDI - Motorproblem

### Turbo
- Turbon kan gå sönder pga oljesvält eller koluppbyggnad
- Symptom: Kraftförlust, ökad avgasrök
- Variabla skovlar kan fastna (rostiga/smutsiga)

### Bränsleinsprutning
- Injektorer kan bli igensatta - ger ojämn tomgång och sämre bränsleekonomi
- Kontrollera bränsleretur på injektorerna vid misstänkt fel

### EGR-ventil
- Blir ofta igensatt av kolbeläggningar
- Ger dålig motorprestanda och motorvarningslampa
- Kan rengöras eller bytas

### DPF (Partikelfilter)
- Kan bli igensatt vid mycket kortkörning
- Kräver regelbunden regenerering (längre körningar på motorväg)
- Symptom: Reducerad effekt, ökad bränsleförbrukning

### Startproblem
- Startmotor kan slitas redan vid 50 000 km
- Vanligare vid kallt väder
- Kontrollera batteri, säkringar och anslutningar

### Växellåda
- Problem med att lägga i backen förekommer
- Kopplingen håller normalt 150 000-200 000 km
- Vid tung last kan kopplingen slitas redan vid 100 000 km

### Oljud från Motor
- Oljeläckage från packningar är vanligt
- Kontrollera oljenivå regelbundet

## Förebyggande Åtgärder

1. **Regelbunden service** enligt schema
2. **Längre körningar** minst en gång i månaden för DPF-regenerering
3. **Kontrollera tätningar** på tak och fönster årligen
4. **Smörj gångjärn och lås** på dörrar och skåp
5. **Vintervård** - töm vattensystemet vid frostfara
